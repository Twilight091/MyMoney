# 💰 MyMoney Pro

A secure, offline-first personal finance tracker built as a Progressive Web App (PWA). No data leaves your device - all information is stored locally using IndexedDB.

## ✨ Features

- **🔒 100% Offline** - Works without internet connection
- **📊 Visual Reports** - Interactive charts for income/expense analysis
- **💸 Transaction Tracking** - Log income and expenses with categories
- **🤝 Loan Management** - Track money lent and borrowed
- **🎯 Budget Planning** - Set monthly budgets with visual progress
- **🌓 Dark/Light Mode** - Toggle between themes
- **💱 Multi-Currency** - Support for USD, EUR, GBP, JPY, INR ,BDT

## 🚀 How to Use

1. Visit the app in your browser
2. Click "Add to Home Screen" when prompted
3. Start tracking your finances!

## 🔒 Privacy

- No server - everything runs in your browser
- No tracking or analytics
- No ads
- Data stays on your device
